問題2：BCC構造の回折に関する解答と解説

問2-1：BCCの単位胞における構造因子

BCCの単位胞中の2つの原子位置をrnで示し、それを使ってFbccを記述せよ。原子の散乱振幅はfとする。

BCCの単位胞における原子位置は以下で表されます：

r1 = (0, 0, 0) [原点の原子]
r2 = (a/2)(1, 1, 1) [体心の原子]

これらを用いて構造因子は：

Fbcc = f[1 + exp(πi(h+k+l))]

解説

  • 原点にある原子からの寄与：f
  • 体心位置の原子からの寄与：f・exp(πi(h+k+l))
  • 位相因子exp(πi(h+k+l))は体心原子による散乱波の位相差を表します

問2-2：Bragg条件下での式の書き換え

問2-1で得られた式について、qがBragg条件を満たす場合について書き換えよ。

Bragg条件下では、構造因子は：

Fbcc = 2f （h+k+lが偶数の場合）
Fbcc = 0 （h+k+lが奇数の場合）

解説

  • exp(πi(h+k+l)) = ±1 となります
  • h+k+lが偶数の場合：exp(πi(h+k+l)) = 1
  • h+k+lが奇数の場合：exp(πi(h+k+l)) = -1
  • これにより、消滅則が導かれます

問2-3：消滅則と回折強度

BCC構造の消滅則を求め、ピークが出る条件での回折強度を示しなさい。

消滅則

  • h + k + l = 偶数の場合のみ回折が生じる
  • h + k + l = 奇数の場合は回折が消滅

回折強度

I = |Fbcc|² = 4|f|² （ピークが出る条件の場合）

物理的意味

  • 体心原子からの散乱波の位相関係により決定される
  • h+k+lが偶数の場合、原点と体心からの散乱波が同位相で強め合う
  • h+k+lが奇数の場合、逆位相で打ち消し合う
  • 4|f|²という値は、2つの原子からの散乱波が完全に同位相で重なった結果
