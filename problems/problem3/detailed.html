散乱波の重ね合わせに関する詳細解説

1. 基本式の導出

散乱波の一般式

F = Σ fn exp(2πiΔn/λ)

パラメータの説明：

  • fn: 原子散乱因子（本問では全て f = 1 として扱う）
  • Δn: 行路差
  • λ: 入射X線の波長（= 1.4Å）

2. Z軸方向の散乱（F14Z）の詳細計算

Step 1: Z座標のグループ化

Z座標値によって原子をグループ化します：

Z座標 原子番号 原子数
0 r0 1個
1.4Å r4, r9 2個
2.8Å r5, r6, r7, r8 4個
5.6Å r10, r11, r12, r13 4個

Step 2: 位相因子の計算

exp(2πiz/λ) の計算：
z = 0: exp(0) = 1
z = 1.4Å: exp(2πi·1.4/1.4) = exp(2πi) = 1
z = 2.8Å: exp(2πi·2.8/1.4) = exp(4πi) = 1
z = 5.6Å: exp(2πi·5.6/1.4) = exp(8πi) = 1

Step 3: 最終式の導出

F14Z = f[1 + 2exp(2πi) + 4exp(4πi) + 4exp(8πi)]
= f[1 + 2 + 4 + 4]
= 11f

3. X軸方向の散乱（F14X）の詳細計算

Step 1: X座標のグループ化

X座標 原子番号 原子数
0 r0, r2, r5, r7, r10, r12 6個
1.4Å r4, r9 2個
2.8Å r1, r3, r6, r8, r11, r13 6個

Step 2: 位相因子の計算

exp(2πix/λ) の計算：
x = 0: exp(0) = 1
x = 1.4Å: exp(2πi·1.4/1.4) = exp(2πi) = 1
x = 2.8Å: exp(2πi·2.8/1.4) = exp(4πi) = 1

Step 3: 最終式の導出

F14X = f[6 + 2exp(2πi) + 6exp(4πi)]
= f[6 + 2 + 6]
= 14f

4. 散乱強度の角度依存性

Debye式の適用

I(q) = Σ Σ fn fm exp(iq·rnm)

ここで：

  • q: 散乱ベクトル（= 4π sinθ/λ）
  • rnm: 原子間ベクトル

小角散乱領域での近似

I(q) ≈ N²[1 - (q²Rg²)/3]
ここで、Rg: 慣性半径
