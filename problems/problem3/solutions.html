問題3：結晶のサイズ効果に関する解析

問3-1：z軸方向の散乱波の重ね合わせ

散乱ベクトルをz軸方向に取ったときの散乱波の重ね合わせF14Zを求める。

z軸方向の散乱を考えると、各原子のz座標による位相差を考慮します：

F14Z = Σf exp(2πiz/λ)
= f[1 + 2exp(2πi·1.4/1.4) + 4exp(2πi·2.8/1.4) + 4exp(2πi·5.6/1.4)]
= f[1 + 2 + 4 + 4]
= 11f

計算の手順：

  1. z = 0の位置：1個の原子からの寄与
  2. z = 1.4Åの位置：2個の原子からの寄与
  3. z = 2.8Åの位置：4個の原子からの寄与
  4. z = 5.6Åの位置：4個の原子からの寄与

これらの寄与を合計すると、全体として11fという結果が得られます。

問3-2：x軸方向の散乱波の重ね合わせ

散乱ベクトルをx軸方向に取ったときのF14Xを求める。

F14X = Σf exp(2πix/λ)
= f[6 + 2exp(2πi·1.4/1.4) + 6exp(2πi·2.8/1.4)]
= f[6 + 2 + 6]
= 14f

x座標位置での原子の分布：

  • x = 0の位置：6個の原子（r0, r2, r5, r7, r10, r12）
  • x = 1.4Åの位置：2個の原子（r4, r9）
  • x = 2.8Åの位置：6個の原子（r1, r3, r6, r8, r11, r13）

各位置での位相因子を考慮すると、全体として14fという結果になります。

問3-3：異なる散乱角での強度比較

2θ = 50°, 55°, 60°, 65°, 70°における散乱強度を比較し、サイズとピーク幅の関係を説明する。

散乱強度の角度依存性

I(θ) = |F(θ)|²

Scherrerの式による解析

Δ(2θ) ≈ λ/(D cosθ)

サイズ効果の特徴：

  • 結晶サイズの有限性によるピークのブロード化
  • 異なる方向（z方向とx方向）での異なるピーク幅
  • 高角側でのピーク分離の向上
  • 粒子サイズとピーク幅の逆比例関係

問3-4：小角散乱領域の解析

2θ = 0°, 5°, 9°での強度を比較し、サイズ効果について説明する。

小角散乱での特徴

I(q) ≈ I(0)exp(-Rg²q²/3)

散乱強度の角度依存性

  • 2θ = 0°付近：
    • 粒子全体の形状を反映
    • 最大強度を示す
  • 角度の増加に伴う強度減衰：
    • Guinierの法則に従う減衰
    • 慣性半径Rgに依存

広角散乱との比較

散乱情報の違い：

  • 小角散乱：粒子の全体サイズと形状に関する情報
  • 広角散乱：結晶構造と結晶子サイズに関する情報
  • 両者の相補的な情報を組み合わせることで、より完全な構造解析が可能
